RoboChrome

Generating a color font

Generate your font via RoboFont as usual.

For the sbix format you need to generate a TTF, it won’t work with OTFs. For the other color formats, the outline format doesn’t matter.

If you click the Export to font button, RoboChrome will know where to look for the font you just generated. RoboChrome adds the color formats you selected, and saves a separate font for each font with a format suffix next to your normal font.

The Adobe/Mozilla and Microsoft formats can live together in one font without interfering with each other. RoboChrome doesn’t support this yet, but you can splice together the different color tables in one font using ttx or similar tools.

Bitmap format options

Hit the Settings... button to open a drawer with more detailed options. For the bitmap formats, you can supply a comma-separated list of sizes to rasterize.

Bitmaps add considerably to the file size of the final font, and are quite slow to rasterize.
