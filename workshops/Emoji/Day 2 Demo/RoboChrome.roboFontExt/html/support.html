Color Font Support in Systems and Applications

All formats can be tried out in the free FontLab Pad application.

Adobe/Mozilla (SVG)

The SVG format is known to work in recent Firefox versions on all platforms. Other applications will display the normal black-and-white glyphs as a fallback.

Microsoft (COLR/CPAL)

Fonts with the COLR/CPAL tables work in Internet Explorer 11 on Windows 8.1. Other applications will display the normal black-and-white glyphs as a fallback.

The Adobe/Mozilla and Microsoft formats can live together in one font without interfering with each other.

Mac OS/iOS (sbix)

The sbix format is supported on Mac OS X 10.7+ and iOS. The color glyphs are only displayed when the fonts are installed in the system, not as web fonts. Google (CBDT/CBLC)

Support for the CBDT/CBLC tables is not implemented yet, and application support is unknown.
