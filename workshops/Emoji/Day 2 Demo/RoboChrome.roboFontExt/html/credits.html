RoboChrome

Credits

RoboChrome was written between 2013 and 2014 by Jens Kutilek at FontShop International in Berlin. He also figured out the Apple color font format and wrote the FontTools module code for it.

It contains code from Behdad Esfahbod’s fork of the Python FontTools, notably the COLR/CPAL and SVG table modules, and the flat Python module by Juraj Sukop for rasterizing the export bitmaps.

The SVG pen code was contributed by Andreas Eigendorf.

Benedikt Bramböck discovered an OpenType-SVG example font, which made implementing SVG export much easier than just by reading the specs.

Inga Plönnigs built our first proper color font with RoboChrome, FF Copa.
