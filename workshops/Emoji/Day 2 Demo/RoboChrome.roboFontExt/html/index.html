RoboChrome

RoboChrome is a RoboFont extension for editing and generating fonts with colored glyphs.

Overview

There are different proposals for how colored glyphs can be stored in a font.

RoboChrome’s model is based on the Microsoft proposal, which allows a black-and-white glyph to be replaced by a stack of colored layer glyphs. Each layer corresponds to one color. Because the format is vector-based, font file sizes stay small.

The layered glyphs are rasterized to PNG images when you choose to export a font in the Apple color font format. These fonts are slow to generate and will take up considerably more file size.

The Adobe/Mozilla proposal allows SVG documents to be stored in a font, and displayed instead of the regular glyphs. In RoboChrome, the SVG document is always derived from the sequence of glyph layers, though native SVGs would allow much more flexibility, e. g. gradients, animation, and bitmap images.
